# WNotes

一个简洁的 Windows 笔记应用。

## 项目概述

WNotes 是一个 Windows 平台桌面笔记应用，采用左右分栏布局，提供简洁高效的笔记管理功能。

### 产品定位
- 轻量级本地笔记应用
- 专注于核心的笔记编辑和管理功能
- 无复杂功能，简单易用

### 核心技术栈

**核心技术栈**：Electron + Vue3 + SQLite + 本地文件目录

**存储核心**：结构化文本（笔记信息）存在 SQLite，二进制图片存在本地目录，分离存储更高效。

**关键依赖**：
- **better-sqlite3**：SQLite 操作
- **Vue3**：前端框架
- **Electron**：桌面应用框架

## 界面设计

### 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  WNotes                                    □ - ×       │
├─────────────────┬───────────────────────────────────────┤
│                 │                                       │
│   笔记列表      │           笔记编辑器                   │
│   (左侧面板)    │           (右侧面板)                  │
│                 │                                       │
│                 │                                       │
└─────────────────┴───────────────────────────────────────┘
```

### 左侧面板 - 笔记列表

**功能组件**:
- 新建笔记按钮（置顶）
- 笔记列表项（垂直排列）
- 删除笔记按钮（每个笔记项）

**列表项显示内容**:
- 笔记标题
- 最后更新时间

**交互行为**:
- 点击笔记项 → 选中该笔记，右侧显示内容
- 点击新建按钮 → 创建新笔记，自动选中
- 点击删除按钮 → 删除该笔记（需确认）

### 右侧面板 - 笔记编辑器

**功能组件**:
- 标题输入框（单行文本）
- 富文本编辑器（支持纯文本 + 图片）
- 图片插入（拖拽和粘贴）

**界面布局**:
```
┌─────────────────────────────────────┐
│ [标题输入框]                         │
├─────────────────────────────────────┤
│                                     │
│         笔记内容区域                 │
│      （支持文字和图片）              │
│                                     │
└─────────────────────────────────────┘
```

**交互行为**:
- 标题编辑 → 实时更新左侧列表
- 内容编辑 → 支持纯文本和图片
- 插入图片 → 支持拖拽和粘贴
- 自动保存 → 内容变化后自动保存
- 未选中笔记时 → 显示提示"请选择或创建笔记"

## 数据结构

### SQLite 数据表

**表名**: `notes`

```sql
CREATE TABLE notes (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  file_path TEXT NOT NULL,
  updated_at INTEGER NOT NULL,
  created_at INTEGER NOT NULL
);
```

### 文件存储结构

```
data/
├── notes/                    # 笔记文件目录
│   ├── note-1.md
│   └── note-2.md
├── assets/
│   └── images/               # 图片资源目录
│       ├── note-1/
│       │   ├── image-1.jpg
│       │   └── image-2.jpg
│       └── note-2/
│           └── image-1.jpg
└── metadata.db               # SQLite 数据库文件
```

### 笔记文件格式

每个笔记保存为 HTML 文件（支持图片显示）：

```html
<h1>笔记标题</h1>

笔记内容...

<img src="D:\AppData\WNotes\data\assets\images\note-1\1234567890.png" alt="图片描述" />
```
